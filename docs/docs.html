Addition

条纹[Strip]特性需要为 table 标签添加table-strip类； 悬浮[Hover]特性需要为 table 标签添加table-hover类。

Name Group ID
XiaoMing Collie 001
XiaoHua Collie 002
XiaoHong Collie 003
XiaoLi Collie 004 
                   
                       <table class="table table-strip table-hover" cellspacing="0" cellpadding="0">
                       ...
                       </table>

                            ...
                            </table>
                       </div>
